# Base Lucky Draw

## App Name

Base Lucky Draw

## Tagline

Draw one winner

## Description

Create a one-winner draw on Base, let wallets join once, and reveal the winner with a clear onchain result.

Character count: 109

## Category

Raffle / community / onchain utility

## Short Pitch

Base Lucky Draw gives communities a fast way to launch a small paid raffle, collect entries onchain, and reveal one winner in a clean mobile flow.

## Production Environment Variables

- `NEXT_PUBLIC_LUCKY_DRAW_CONTRACT_ADDRESS`: deployed draw contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
